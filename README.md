# Spatial IHC Feature Lab

A reproducible computational pathology repository demonstrating how to engineer spatial features from multiplex imaging data, train interpretable machine learning models, and evaluate predictive markers.

## Project Goals

* **Primary Goal:** Spatial feature engineering and classical machine learning. The core workflow utilizes the MIBI-TNBC dataset to convert cell coordinates, phenotypes, and marker intensities into engineered biological features (distances, neighborhood densities, graphs) to train interpretable prediction models.
* **Secondary Goal:** Computer Vision (CV) module. Utilizing the MIHIC dataset to build a supporting image classification pipeline, practicing classical image feature extraction and small-scale CNN transfer learning.

*Disclaimer: This is a portfolio and learning project, not a clinical product.*
